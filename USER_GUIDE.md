# 用户使用指南

## 功能介绍

App ROI数据分析系统是一个用于分析移动应用广告投放ROI（投资回报率）趋势的可视化工具。系统支持多时间维度的ROI数据展示，帮助您了解广告投放效果。

## 数据说明

系统支持以下8个时间维度的ROI数据：

| 维度 | 说明 |
|------|------|
| 当日ROI | 投放当天的投资回报率 |
| 1日ROI | 投放后1天的累计投资回报率 |
| 3日ROI | 投放后3天的累计投资回报率 |
| 7日ROI | 投放后7天的累计投资回报率 |
| 14日ROI | 投放后14天的累计投资回报率 |
| 30日ROI | 投放后30天的累计投资回报率 |
| 60日ROI | 投放后60天的累计投资回报率 |
| 90日ROI | 投放后90天的累计投资回报率 |

## 界面说明

### 1. 页面标题区

页面顶部显示当前选中的应用名称和图表标题：
- 主标题："App-X - 多时间维度ROI趋势"
- 副标题："(7日移动平均)"
- 数据范围：显示当前筛选的数据日期范围

### 2. 筛选器

第一行筛选器（从左到右）：

#### 用户安装渠道
- **选项**：Apple
- **说明**：固定为Apple渠道

#### 出价类型
- **选项**：CPI（每次安装成本）
- **说明**：数据集中的出价类型统一为CPI

#### 国家地区
- **选项**：美国、英国
- **说明**：选择特定的投放国家进行分析

#### APP
- **选项**：App-1、App-2、App-3、App-4、App-5
- **说明**：选择要分析的应用

#### 日期范围
- **开始日期**：选择数据起始日期
- **结束日期**：选择数据结束日期
- **默认范围**：2025-04-13 至 2025-07-12

### 3. 控制器

#### 数据显示模式
- **显示移动平均值**：显示7日移动平均后的数据，平滑波动
- **显示原始数据**：显示原始ROI数据

#### Y轴刻度
- **线性刻度**：Y轴使用线性比例，适合大多数情况
- **对数刻度**：Y轴使用对数比例，适合数据范围跨度大的情况

## 图表操作

### 基本操作

1. **鼠标悬停**：将鼠标移动到数据点上，显示具体的ROI数值
2. **图例点击**：点击图表下方的图例，可以显示或隐藏对应的数据线
3. **滚轮缩放**：部分浏览器支持使用滚轮缩放图表

### 数据线说明

图表中显示8条不同颜色的趋势线，每条线代表一个时间维度的ROI：
- 蓝色：当日ROI
- 橙色：1日ROI
- 绿色：3日ROI
- 红色：7日ROI
- 紫色：14日ROI
- 棕色：30日ROI
- 粉色：60日ROI
- 灰色：90日ROI

### 100%回本线

红色水平线标记100%回本位置：
- 超过此线：表示已回本并有盈利
- 低于此线：表示尚未回本

### 0%数据处理

系统会智能处理0%数据：
- **真实0%**：数据点正常显示，悬停提示显示"实际ROI为0%"
- **日期不足0%**：距离数据截止日期不足对应天数，图表中显示为断点，悬停提示显示"数据不完整"

## 常见问题

### Q: 为什么某些数据点为0%？

A: 0%有两种可能：
1. **真实0%**：该日期的实际ROI确实为0%（无收益或亏损）
2. **日期不足0%**：该日期到数据截止日期的间隔不足对应的ROI统计周期

例如：如果某条记录日期是2025-07-01，数据截止是2025-07-12，距离只有11天，则30日、60日、90日ROI都会显示为0%（日期不足）。

### Q: 为什么图表中有断点？

A: 断点表示"日期不足"的0%数据，系统自动断开连接线，避免误导。完整的数据会正常连接显示。

### Q: 移动平均是如何计算的？

A: 7日移动平均取当前日期和前6天共7个数据点的平均值。图表中前6天没有完整数据，所以显示为空白。

### Q: 如何选择合适的Y轴刻度？

A: 
- **线性刻度**：适合ROI在相似范围内波动（如都在0-200%之间）
- **对数刻度**：适合ROI范围跨度大（从接近0到数百或数千）

### Q: 图例中的数据线与显示不符？

A: 检查：
1. 是否点击了图例隐藏了某些数据线
2. "数据显示模式"是否设置为"显示移动平均值"
3. 筛选条件是否正确

## 数据安全

- 所有数据存储在本地数据库中
- CSV导入时会跳过重复数据
- 数据不会被上传到第三方服务
